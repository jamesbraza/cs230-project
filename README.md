## Developers

This project was developed using Python 3.10.

### Getting Started With `pre-commit`

```bash
python -m venv venv
source venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt -r requirements-qa.txt
pre-commit install
```
